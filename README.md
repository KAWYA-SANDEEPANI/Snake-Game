Snake Game
A simple classic Snake Game built using **HTML, CSS, and JavaScript**.  
The player controls the snake, eats food, grows longer, and tries to get the highest score without hitting the walls or itself.

Features
- Smooth snake movement
- Random food spawning
- Score tracking
- Game over detection
- Restart button
- Simple and clean UI


 Technologies Used
- HTML5 (Canvas)
- CSS3
- JavaScript (Vanilla JS)

How to Play
- Use **arrow keys** to control the snake:
  - ⬆️ Up
  - ⬇️ Down
  - ⬅️ Left
  - ➡️ Right
- Eat food to grow and increase score
- Avoid hitting walls or yourself
